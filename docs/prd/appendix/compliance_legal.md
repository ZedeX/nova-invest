## 1. Regulatory Positioning [B]

### 1.1 Legal Positioning of nova-invest

**Key positioning decision**: nova-invest is positioned as **"Publisher + Tech Platform"**, not "Registered Investment Adviser (RIA)".

| Role | Bearing | Description |
|---|---|---|
| Investment Adviser (RIA) | ❌ | Does not provide personalized investment advice |
| Broker-Dealer | ❌ | Phase 1 paper trading only |
| Exchange (ATS) | ❌ | Does not match real trades |
| Content Publishing Platform | ✅ | User-generated Playbook content |
| Technology Tool Provider | ✅ | Provides Agent + DSL + backtest tools |

### 1.2 US SEC-related Exemptions

**Investment Advisers Act of 1940** exemption clauses [KNOWN]:

- **Section 202(a)(11) Publisher Exemption**: Content periodically published publicly (such as Playbook community) does not constitute "investment advice", provided that:
  - Content is non-personalized
  - Does not directly drive trade execution
  - Publicly published

- **Section 203(b)(3) de minimis exemption**: Customers < 15 in 12 months can be exempt from registration. nova-invest users, even if paying, do not constitute "client relationship" because content is fully self-service

### 1.3 User Agreement Key Points

**Terms of Service key clauses**:

1. **"Not Investment Advice" statement**: All AI-generated content clearly marked "not investment advice"
2. **At your own risk**: Users bear their own profits/losses from trading decisions made using this tool
3. **Prohibited behavior**: Prohibit using this tool for insider trading, market manipulation
4. **Data license**: Strategies / Playbooks uploaded by users default to CC-BY-4.0 license for community
5. **Service availability**: No guarantee of 99.99% availability (free Cloudflare tier)
6. **AI content disclaimer**: AI may produce incorrect information, users must verify

## 2. AI-Generated Content Disclaimer [B]

It is not investment advice. Verify all data with primary sources before making
any financial decisions. See full disclaimer: /legal/ai-disclaimer

### 2.2 Hallucination Risk Disclosure

**Must be checked at user registration**:

```
[ ] I understand that AI may produce incorrect or fabricated information
("hallucinations"). I will independently verify all numeric data, citations,
and claims before acting on them.
```

### 2.3 Citations and Source Annotation

- Every AI response must include a citations array (even if empty)
- Numeric fields must be extracted from RAG (LLM free generation not allowed)
- Users can click citation in UI to jump to original text for verification

## 3. Simulated Trading Is Not Investment Advice [B]

### 3.1 Paper Trading Statement

```
nova-invest's Paper Trading feature is a simulation only. Performance in
simulation does not guarantee future results. Real-money trading involves
risk of loss, including the loss of principal. Paper trading results may
differ materially from real-money results due to slippage, market impact,
and other factors.
```

### 3.2 Backtest Limitations Disclosure

**Every backtest report must display**:

```
⚠️ Backtest Limitations:
- Past performance does not guarantee future results
- Backtest may be subject to survivorship bias
- Backtest may be subject to look-ahead bias if data is misused
- Slippage and commission assumptions may not reflect real-world conditions
- In-sample optimization may overfit; out-of-sample results are more reliable
```

### 3.3 Strategy Risk Disclosure

**Every Playbook must fill in `narrative.risks` field at publish time** (see Epic 08), with at least 1 risk disclosure.

## 4. Privacy Data Rules [B]

### 4.1 Data Collected

| Data Type | Purpose | Shareable |
|---|---|---|
| Email | Account login | Not public |
| Position info | Personalized service | Strictly confidential |
| Conversation history | Ask Agent context | User can export/delete |
| Credit card | Payment (Phase 2) | Stripe processed, not stored locally |
| Browsing behavior | Recommendation optimization | Anonymized aggregate |

### 4.2 GDPR / CCPA Compliance

**User rights**:

- **Access right**: User can export all personal data (JSON format)
- **Deletion right**: User can delete account + all data, completed within 30 days
- **Correction right**: User can modify personal profile
- **Data portability right**: Export standard JSON format
- **Objection right**: User can opt out of personalized recommendations

### 4.3 Data Storage

- **D1**: User metadata / positions / conversation history
- **R2**: Playbook YAML / Mock K-lines (public data)
- **Not stored**: Credit cards (Stripe), SSN, bank accounts

## 5. User Agreement Key Points [B]

### 5.1 ToS Key Clauses

1. **Acceptance of terms**: Registration constitutes acceptance
2. **Eligibility**: User must be 18+ and legal in their jurisdiction
3. **Account security**: User is responsible for account activity
4. **Prohibited behavior**:
   - Insider trading
   - Market manipulation
   - Reverse engineering API (Mock data can be public but LLM prompts are copyright-protected)
   - Scraping others' Playbooks for commercial use
5. **Intellectual property**:
   - nova-invest platform code belongs to nova-invest
   - User Playbooks belong to users (default CC-BY-4.0)
6. **Service changes**: Reserve right to modify/terminate services
7. **Limitation of liability**: Max liability = amount paid by user in past 12 months
8. **Dispute resolution**: Mediation first, then arbitration (American Arbitration Association AAA)
9. **Applicable law**: US Delaware state law

### 5.2 Privacy Policy Key Clauses

1. **Collection scope**: See 4.1
2. **Use scope**: Only for providing services
3. **Sharing scope**: Do not sell user data to third parties
4. **Cookies**: Only necessary cookies + Google Analytics (user can opt out)
5. **Data retention**: Account deletion completely purged after 30 days
6. **Children**: Do not accept users under 18
7. **International users**: Users outside US transfer data to US, must consent
